Working with AI

# Think → AI → Think
I think, I delegate, I think again. The result is only as sharp as the question I ask and the judgment I bring to it.

## The right foundation
### Shared infrastructure
Custom skills, hooks, and plugins versioned in GitHub. The team pulls the same toolchain. Improvements compound instead of getting trapped on someone's laptop.

### Aligned with tech
Tool choices made with the engineers who ship the code. Prototypes use the real component library. Handoff is clean because the tracks were never separate.

### Compliance built in
Risk assessments, DPAs, GDPR and IP respected on input and output, sandboxed models where the data demands it.

### AI as a user
The newest user segment I design for. Notes in markdown. Structured project folders. READMEs that double as onboarding context. A model should be able to pick up a project cold, the same way a new teammate would.
